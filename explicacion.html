Como se Creo Esta Web

Una explicacion sencilla y visual de como la Inteligencia Artificial ayudo a crear este sitio web. Sin tecnicismos, prometido.

Imaginalo asi...

Crear esta web con IA es como tener una conversacion con un arquitecto muy talentoso. Tu le dices lo que quieres ("quiero una casa con 3 habitaciones y un jardin"), y el arquitecto no solo dibuja los planos, sino que tambien construye la casa al instante.

En nuestro caso, el "arquitecto" es Claude, una IA de Anthropic, y la "casa" es esta pagina web.

El Proceso Paso a Paso

1. La Conversacion Inicial

Todo empezo con una conversacion. Le explicamos a la IA que queriamos: una web con estilo MIT, que mostrara informacion de los participantes, que fuera sencilla y se viera bien en moviles.

"Queremos una web estatica con estilo MIT, que presente a los participantes del programa de forma atractiva..."

2. Entendiendo el Contexto

La IA leyo los perfiles de todos los participantes (CVs, descripciones) para entender quienes sois y crear contenido relevante y personalizado.

Esto es como cuando un disenador estudia a su cliente antes de empezar.

3. Creacion del Diseno

Con toda la informacion, la IA creo los estilos visuales (colores MIT, tipografias elegantes) y la estructura de las paginas.

HTML CSS Responsive

4. Escritura del Codigo

La IA escribio todo el codigo necesario: la estructura de las paginas (HTML), los estilos visuales (CSS), y organizo todo de forma profesional.

Esto que normalmente tomaria horas o dias, la IA lo hace en minutos.

5. Publicacion en Internet

Finalmente, la IA subio todo a GitHub (un servicio de almacenamiento de codigo) y activo GitHub Pages para que la web este disponible en internet.

Y voila! La web esta en linea para que todos la vean.

Las Tecnologias Utilizadas

🤖

Claude (IA)

La inteligencia artificial de Anthropic que "penso" y escribio todo el codigo.

📄

HTML

El "esqueleto" de la web. Define la estructura y el contenido.

🎨

CSS

La "ropa" de la web. Define colores, fuentes y como se ve todo.

🐙

Donde vive el codigo y donde se publica la web para que todos la vean.

Un Ejemplo de la Conversacion

Asi es como se ve una conversacion tipica con la IA durante el desarrollo:

HUMANO
Quiero una web con estilo MIT que presente a los participantes del programa.
CLAUDE (IA)
Entendido. Voy a leer los perfiles de los participantes para conocer al grupo. Que tipo de informacion quieres mostrar de cada persona?
HUMANO
Algo resumido: nombre, rol profesional y pais. Tu decides el diseno.
CLAUDE (IA)
Perfecto. Voy a crear tarjetas elegantes con animaciones sutiles. Comenzare con el CSS para el estilo MIT y luego las paginas HTML.

Lo Mas Importante

Esta web demuestra que la IA no reemplaza a las personas, sino que las potencia. Un humano tuvo la idea y dio las instrucciones; la IA ejecuto con precision y velocidad.

Es la combinacion perfecta: creatividad humana + capacidad de ejecucion de la IA.

Fase II: El OrquestadorAgent

En la segunda fase, anadimos un agente conversacional que puede interactuar contigo. Pruebalo haciendo clic en el boton rojo de la esquina inferior derecha.

💬

Smalltalk

El agente entiende saludos, despedidas y agradecimientos en espanol.

🧹

Detector de Basura

Identifica mensajes sin sentido como "asdfghjk" y responde apropiadamente.

🚀

JavaScript

Corre directamente en tu navegador, sin necesidad de servidores externos.

🤝

Preparado para Delegar

El orquestador puede delegar tareas a otros agentes especializados.

Como funciona el OrquestadorAgent?

Es como un recepcionista muy educado. Cuando llegas, te saluda; cuando te vas, se despide; cuando agradeces, responde amablemente. Y si dices algo sin sentido... te lo hace saber con humor.

Fase III: Sistema Multiagente

Ahora tenemos dos agentes trabajando juntos. El OrquestadorAgent detecta cuando necesitas hacer calculos y delega la tarea al MatematicasAgente.

Imaginalo asi...

Es como una empresa con un recepcionista (OrquestadorAgent) y un contable (MatematicasAgente). Cuando llegas y pides hacer una suma, el recepcionista te atiende amablemente y llama al contable para que haga el calculo. Luego te devuelve el resultado.

🎯

OrquestadorAgent

Recibe todos los mensajes y decide que hacer con ellos.

🧮

MatematicasAgente

Especialista en /suma, /resta, /multiplicacion y /division.

🔄

Delegacion

El orquestador detecta comandos y los envia al agente correcto.

✨

Respuesta Unificada

El usuario siempre habla con el orquestador, sin saber quien calcula.

USUARIO
/suma 15 27
ORQUESTADOR → MATEMATICAS → ORQUESTADOR
🧮 15 + 27 = 42
Calculado por MatematicasAgente
